# HOA Integration

Status: Approved for Planning

## Core Concept

HOA is not a separate application.

HOA is a property mode within the same Barbarosa / HomeOS platform.

The same core system supports:

* Single Family Homes
* Condos
* Apartment Units
* HOA Communities
* Large Apartment Buildings
* Commercial Properties

The equipment, documents, photos, history, and health systems remain the same.

Only the account type and management structure change.

---

## Contact System

The contact button changes based on account type.

Homeowner:

* Contact Service Provider

HOA Resident:

* Contact HOA

Apartment Resident:

* Contact Property Manager

Commercial Tenant:

* Contact Building Management

## HOA Home Screen

Additional HOA features:

* Community Notices

* HOA Documents
* Maintenance Requests
* Approved Vendors
* Community Announcements

---

## Maintenance Request Flow

Resident selects:

* Plumbing
* HVAC
* Electrical
* Roofing
* Landscaping
* Pool
* General Maintenance

Request is sent directly to HOA management.

HOA decides whether to:

* Dispatch in-house staff
* Assign vendor
* Request additional information
