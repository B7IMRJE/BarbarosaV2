BARBAROSA

190 - SERVICE REQUEST SYSTEM

VERSION 1

==================================================
PURPOSE
=======

The Service Request System provides homeowners with a
simple method for requesting service, reporting problems,
requesting inspections, and handling emergencies.

Purpose:

* Simplify communication
* Improve emergency response
* Connect homeowners to providers
* Create automatic Home History records
* Improve diagnostics before arrival

==================================================
